## CloudFormation Infrastructure

Added AWS infrastructure automation using CloudFormation.

### Templates Added

- VPC creation
- Security Groups
- EC2 provisioning
- VPC Interface Endpoints for AWS Systems Manager

### Deployment Order

1. VPC
2. Security Groups
3. EC2 (frontend and backend)
4. VPC Endpoints

### AWS Services Used

- Amazon VPC
- EC2
- IAM
- Systems Manager (SSM)
- CloudFormation
